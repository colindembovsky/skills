# Context Packet Plugin

Install the `context-packet` skill with GitHub CLI:

```bash
gh skill install colindembovsky/skills context-packet
```

To pin a specific version:

```bash
gh skill install colindembovsky/skills context-packet@v0.1.2
```

The plugin manifest is at `.github/plugin/plugin.json`; the skill source is at
`skills/context-packet/`.
